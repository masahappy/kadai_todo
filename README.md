# AIToDo

AIを活用したタスク管理アプリです。

## 概要

AITodoは、日々のタスクを効率的に管理するためのWebアプリケーションです。
未完了のタスクと今週の予定を入力すると、AIが効率的なスケジュールを提案してくれます。
学生から社会人まで、幅広い方に活用していただけます。

## 機能

- タスクの追加・削除・完了管理
- 締切日・優先度・カテゴリの設定
- 今週の予定入力（曜日ごと・体調）
- AIによるスケジュールアドバイス

## 使用技術

| 技術 | 用途 |
|---|---|
| HTML / CSS / JavaScript | フロントエンド |
| Groq API（llama-3.3-70b-versatile） | AIアドバイス機能 |
| Netlify Functions | APIキーの安全な管理 |
| Netlify | ホスティング・デプロイ |
| localStorage | データの保存 |

## 工夫した点

- **AIアドバイス機能**：未完了のタスクと今週の予定・体調をもとに、AIが具体的なスケジュールを提案します。単なるタスク管理にとどまらず、ユーザーの状況に合わせた提案ができる点が特徴です。
- **セキュアな実装**：APIキーをフロントエンドに直接書かず、Netlify Functionsを用いてサーバーサイドで安全に管理しています。

## 苦労した点

- AIの実装にあたっての試行錯誤。APIキーの安全な管理方法や、モデルの廃止への対応など、多くの課題を一つひとつ解決しながら開発を進めました。
